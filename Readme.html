Five Nights at Freddy's Gimmick


By Greenhammerbro


This gimmick recreates the FNAF mode in which the player must survive for a period of time (12AM to 6AM) in order to finish the level (not necessary ending the level, it teleports the player out of the “FNAF levels”).

Note: I highly recommend have at least some experience with ASM due to the stabilizability being complex, and that you'll likely have to make changes to this. It features viewing other levels, layer 3 map interface, and more.

What you need:
  • [Tool] Gopher Popcorn stew
  • [Tool] Uberasm tool
  • [Custom Block] Controller Execute Once blocks


I did include a sample level and other pre-done “assets” to get an idea on how this works. Key facts:
  • Timer: A simple 16-bit frame counter that increments. Hits a certain value, and teleports the player to a specific level. This is run by Uberasm tool code.

  • Power:A 24-bit (3 bytes) value that decrements. Note that this is a fixed-point value; Byte 0 is a fraction out of 256, and the other two are the values to be read, therefore its $xxxx.xx.
